BindMenu PRO v2.0

Autor: CodeKernel

Descrição:
BindMenu PRO é um script para SAMP (San Andreas Multiplayer) com MoonLoader + ImGui que permite criar,
gerenciar e ativar comandos personalizados (binds) de forma fácil e rápida, sem necessidade de atalhos de teclado.

Funcionalidades:
- Menu ImGui interativo, aberto via comando /bindar
- Adicionar novos comandos/binds manualmente
- Executar comandos clicando nos botões do menu
- Remover binds de forma simples
- Salvar e carregar binds automaticamente usando arquivo de configuração (.ini)
- Créditos explícitos no menu
- Compatível com servidores que permitem binds manuais

Como usar:
1. Digite /bindar no chat do SAMP para abrir o menu
2. Adicione comandos no campo de texto e clique "Adicionar Bind"
3. Clique nos botões dos comandos para enviá-los automaticamente no chat
4. Remova binds clicando no botão "Remover"
5. Seus binds são salvos automaticamente e carregados na próxima vez que o script iniciar

Requisitos:
- MoonLoader
- Lua + ImGui + inicfg

Aviso:
Este script é **legal e seguro** para servidores que permitem binds manuais. Não altera memória ou velocidade do jogo.
Certifique-se de respeitar as regras do servidor.
